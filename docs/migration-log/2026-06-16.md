# 2026-06-16

## CI 修复

- 修复 GitHub Actions Flutter CI 在 Linux runner 上错误调用 `powershell` 导致找不到命令的问题，改为在 `pwsh` shell 中直接执行 `./tool/run_daily_checks.ps1`。
- 为 Flutter CI 显式固定 `PUB_HOSTED_URL=https://pub.dev` 和 `FLUTTER_STORAGE_BASE_URL=https://storage.googleapis.com`，避免 CI 继承本地镜像源配置。
- 将仓库内 `pubspec.lock` 与 `packages/lucent_openapi/pubspec.lock` 的 hosted 源从 `pub.flutter-io.cn` 切回 `pub.dev`，避免 `flutter pub get` 在 advisory 响应解析阶段失败。
