  - 重复成分检测从 `normalizedIngredientTokens` 改为 `canonicalIngredientKeys`，实现对乙酰氨基酚 ↔ acetaminophen 等跨语言重复识别。

## Desktop Shell SaaS 骨架改造：PRD 与实施计划确认

桌面端从“有侧边栏的移动端放大版”向“SaaS 仪表盘平台感”迈出第一步。本次仅完成 PRD 与实施计划文档落地，代码改造后续执行。

### 需求摘要

- 桌面端侧边栏支持折叠：232px（图标+文字）↔ 72px（仅图标+Tooltip）。
- Tab 常驻桌面：二级页面不再覆盖 Shell，而是在右侧内容区内嵌展示。
- 手机端行为保持不变：二级页面仍全屏，底部 5 个 Tab 不变。
- 折叠状态本地持久化，默认展开。
- 使用 GoRouter 的 `StatefulShellRoute` 实现嵌套路由与 URL 状态。

### 关键设计决策

- 折叠按钮放在侧边栏顶部品牌行 Logo 旁。
- 动画采用 200ms easeOutCubic，标签文字用 `AnimatedOpacity` 避免截断突兀。
- 桌面端断点保持 `AppBreakpoints.desktop = 1200px`。
- 路由分类：
  - **全屏（保留）**：`/login`、`/register`、`/forgot-password`、`/account*`。
  - **内嵌到 Shell 内容区**：`/medicine/*`、`/record/*`、`/mine/*`、`/settings/*`、`/assistant`、`/notifications/*`。
- 隐藏 Branch（settings/assistant/notifications）不参与底部导航高亮。

### 新增文档

- `plans/desktop-shell-saas-layout-prd.md`：需求、验收标准、风险。
- `plans/desktop-shell-saas-layout-plan.md`：分 Phase 实施计划、回滚策略、时间预估。

## Phase 2 实施：路由改造为 StatefulShellRoute

将原本平铺的二级页面路由迁移到 GoRouter 的 `StatefulShellRoute.indexedStack`，使桌面端侧边栏在二级页面保持可见，手机端隐藏分支仍全屏。

### What changed

- `lib/app/router.dart`
  - 引入 `StatefulShellRoute.indexedStack`，`builder` 返回 `ShellPage(navigationShell: navigationShell)`。
  - 8 个 Branch，顺序与 `ShellBranch` 对齐：
    - 可见 Branch：`/`（Today）、`/record`、 `/medicine`、 `/report`、 `/mine`
    - 隐藏 Branch：`/settings`、 `/assistant`、 `/notifications`
  - 将原有 `/medicine/*`、`/record/*`、`/mine/*`、`/settings/*`、`/assistant`、`/notifications/*` 作为子路由挂到对应 Branch 下。
  - 保留原有 `_slidePage` / `_fadePage` 转场：Branch 根页面使用 `NoTransitionPage`，子页面继续使用 `_slidePage`。
  - auth 与 account 管理路由仍保持顶层全屏。
- 新增 `lib/features/shell/presentation/shell_branch.dart`
  - 定义 8 个 Branch 索引及 `isVisible` 判断，便于 Shell 区分可见 Tab 与隐藏分支。
- `lib/features/shell/presentation/shell_page.dart`
  - 构造函数接收可选的 `StatefulNavigationShell? navigationShell`。
  - 当 `navigationShell` 不为 null 时，从 `navigationShell.currentIndex` 读取当前分支，并通过 `goBranch` 切换；为 null 时回退到旧的 `shellProvider` 行为，保持现有 widget 测试可用。
  - 桌面端始终渲染侧边栏 + 内容区；手机端处于隐藏分支时（`currentIndex >= 5`）不显示底部导航栏，实现全屏效果。
  - 侧边栏 Settings / Help 动作改为 `goBranch` 到对应隐藏分支。
