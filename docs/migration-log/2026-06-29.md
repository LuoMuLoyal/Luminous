## Widget 目录按职责拆分子目录

### 约定

所有 feature 的 `widgets/` 目录统一按以下分类拆分子目录：

| 子目录 | 内容 |
|--------|------|
| `views/` | 页面骨架：`*_dashboard_view.dart`、`*_skeleton_view.dart` |
| `sections/` | 页面内功能区块：`*_section.dart`、`*_grid.dart`、`*_strip.dart` |
| `dialogs/` | 弹窗：`*_dialog.dart` |
| `shared/` | 复用小组件 + 辅助：`*_components.dart`、`*_copy.dart`、`*_shared*.dart` |
| `forms/` | 表单字段组合（仅 record） |
