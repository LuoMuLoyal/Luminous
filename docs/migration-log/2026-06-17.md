## 导出产品化第一轮

- 在 `data_export_controller.dart` 中抽出统一的 `DataExportUiStatus`，把导出状态收口为 `idle / requested / processing / completed / completedLinkMissing / failed / unavailable`。
- Report 导出点击后的反馈改为真实生命周期文案：
  - 已提交
  - 处理中
  - 已完成并尝试打开
  - 已完成但下载链接缺失
  - 失败
  - 当前环境暂不可用
- Settings 导出行改为展示与 Report 同源的最新导出状态，不再只用粗粒度旧标签。
- Report 导出卡片现在会在匹配当前导出类型时直接显示最近一次状态，例如“处理中”“已完成”“暂不可用”，而不是永远只显示静态副标题。
- “导出入口还没接通” 的旧文案已移除，改成更符合真实现状的“当前环境暂时无法生成这个导出文件”。
